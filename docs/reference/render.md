# Render a glyph spec as an htmlwidget

Render a glyph spec as an htmlwidget

## Usage

``` r
render(spec, width = NULL, height = NULL)
```

## Arguments

- spec:

  A glyph_spec, glyph_layout, or glyph_compiled

- width:

  Widget width

- height:

  Widget height

## Value

An `htmlwidget` object that renders the visualization in an HTML viewer
