# Add a title, subtitle, or caption

Add a title, subtitle, or caption

## Usage

``` r
titles(spec, title = NULL, subtitle = NULL, caption = NULL)
```

## Arguments

- spec:

  A glyph_spec

- title:

  Main title

- subtitle:

  Subtitle (below title)

- caption:

  Caption (bottom of plot, for source attribution)

## Value

Modified `glyph_spec` object with updated title metadata
