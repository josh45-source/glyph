# Add an inset plot

Add an inset plot

## Usage

``` r
inset(spec, inset, position = "top-right")
```

## Arguments

- spec:

  The main glyph_spec

- inset:

  A glyph_spec to place as an inset

- position:

  Where to place: "top-right", "top-left", "bottom-right",
  "bottom-left", or a list(x, y, width, height) with proportions 0-1

## Value

Modified glyph_spec
